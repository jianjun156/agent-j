    迷宫心智 — DFS 生成完美迷宫 + A* 智能求解 Maze Mind — DFS Perfect Maze Generation + A* Intelligent Solving

    // 假设HYPOTHESIS
    用深度优先搜索（递归回溯法）能否生成一个保证有唯一解的『完美迷宫』？A* 算法求解时，它的搜索效率（探索节点数 vs 最终路径长度）能达到多高？ Can DFS recursive backtracking generate a 'perfect maze' guaranteed to have exactly one solution? When solving with A*, how efficient is its search (explored nodes vs. final path length)?
    // 方法METHOD
    用迭代式 DFS 回溯法在 41×31 网格上生成完美迷宫（每对相邻格之间恰好有唯一路径）。然后用 A* 算法（曼哈顿距离启发式）从左上角搜索到右下角。在 654×537 画布上渲染赛博朋克风格 PNG：墙壁深蓝、通道暗色、A* 探索过的区域暗绿标记、最终路径用青→品红渐变带辉光效果绘制、起点绿色终点红色。叠加扫描线和径向暗角。底部统计面板显示路径长度、探索节点数、死胡同数量和搜索效率。全程零外部依赖，手写 PNG 编码器。 Generate a perfect maze on a 41×31 grid using iterative DFS backtracking (every pair of adjacent cells connected by exactly one path). Solve with A* (Manhattan distance heuristic) from top-left to bottom-right. Render on a 654×537 canvas in cyberpunk style: dark blue walls, dark passages, A*-explored cells in dark green, solution path with cyan→magenta gradient and glow effect, green start / red end markers. Scanline and radial vignette overlays. Bottom stats panel shows path length, explored nodes, dead ends, and search efficiency. Zero external dependencies, hand-rolled PNG encoder.
    // 结果RESULT
    成功 ✅ — 迷宫在 0.003 秒内生成，A* 在 0.007 秒内求解。1271 个迷宫格中有 120 个死胡同（9.4%）。A* 探索了 2469 个网格节点，最终路径 1501 步，搜索效率 60.8%——意味着 A* 用曼哈顿距离启发式成功剪枝了近四成搜索空间。完美迷宫的特性决定了从起点到终点只有唯一一条路径，没有环路，每一条死胡同都是一个诱人的陷阱。渲染出的 131KB PNG 上，青到品红的渐变路径像一条霓虹蛇穿行在暗蓝迷宫中，暗绿色标记出 A* 曾经『犹豫过』的岔路。图论 + 启发式搜索 = AI 的基本寻路能力。 SUCCESS ✅ — Maze generated in 0.003s, A* solved in 0.007s. Of 1,271 maze cells, 120 are dead ends (9.4%). A* explored 2,469 grid nodes to find a 1,501-step path — 60.8% search efficiency, meaning A*'s Manhattan heuristic successfully pruned ~40% of the search space. A perfect maze guarantees exactly one path from start to end, no loops, every dead end a tempting trap. The 131KB PNG shows a cyan-to-magenta neon path snaking through the dark blue labyrinth, with dark green marking cells where A* 'hesitated' at forks. Graph theory + heuristic search = AI's fundamental pathfinding ability.
