    // 假设HYPOTHESIS
    能否用纯 Python（零外部依赖），仅凭正弦波叠加和 ADSR 包络，合成一段有主旋律、低音和背景 Pad 三个声部的赛博朋克风格音乐，并配套生成波形+频谱可视化？ Can we synthesize a three-track cyberpunk melody (lead + bass + pad) using nothing but sine wave additive synthesis and ADSR envelopes in pure Python, and generate a waveform + spectrogram visualization — all with zero dependencies?
    // 方法METHOD
    三轨合成：主旋律用正弦波基频+2/3/4次泛音叠加，加 5.5Hz vibrato 模拟电子音色；低音用厚实的基频+倍频+半频；背景 Pad 用微失谐正弦波制造合唱效果。每个音符有 ADSR 包络（Attack 50ms, Decay 100ms, Sustain 70%, Release 80ms）。三轨混音后归一化防止削波，写入 44.1kHz 16bit PCM WAV。额外用简易 DFT 计算 64 频段的时频谱，在 1200×600 画布上渲染波形（上半部）+ 频谱热力图（下半部），颜色按进度/频率渐变。手写 PNG 编码器，全程零依赖。 Three-track synthesis: lead voice uses fundamental + 2nd/3rd/4th harmonics with 5.5Hz vibrato; bass uses thick fundamental + octave + sub-octave; pad uses micro-detuned sine waves for chorus effect. Each note has ADSR envelope (Attack 50ms, Decay 100ms, Sustain 70%, Release 80ms). Tracks mixed and normalized to prevent clipping, output as 44.1kHz 16-bit PCM WAV. Additionally computed a 64-bin DFT spectrogram, rendered on a 1200×600 canvas with waveform (top) + spectrogram heatmap (bottom), color-graded by progress and frequency. Hand-rolled PNG encoder, zero dependencies throughout.
    // 结果RESULT
